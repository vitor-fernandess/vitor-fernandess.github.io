Class vs ID

IDs and classes are very similar and very different at the same time.

They are both used to mark up elements and make our code more efficient and cleaner.

The differences are that IDs are always used to mark up only one element in a page, and classes are used to mark up multiple elements within a page.

Example of a Class declaration


Classes are denotated by a period, followed by the class name.
Classes can be assigned to multiple elements trhoughout the website because they are not unique.

ID marks a specific element.
Each element must only have one ID, and each page can only have one element with that specific tag.
