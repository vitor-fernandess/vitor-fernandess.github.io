Javascript and the dom

HTML provides the structure and content, CSS handles the presentation and design, and Javascript adds interactivity and functionality.

Together, HTML, CSS and Javascript collaborate to create a fully functional and visually appealing web experience.

Javascript


Javascript is a programming language that allows our Website to complete complex features on the web page, by enabling us to dynamically update content, for example displaying timely content updates, interactive, maps animated graphics, etc.

css


CSS( Cascade style sheet) is a rule-based language responsible for specifying how documents are presented to users. CSS can be used for text styling for example, changing the color of texts like paragraphs, headings and links. It can also create a layout for example turning a single column of text into a layout.

HTML


HTML (Hypertext markup language) is a standard markup language for web page creation. HTML is responsible for the main structure of the website. It allows the creation of paragraphs, headings, images and also links from other webpages or file documents.

JavaScript workflow


Workflow is the order in which statements are executed in a program. For example the way that we read the newspaper, left-to-right and top-to-bottom.

Loops


Loops are set to repeat a similar, or the same code a number of times depending on the condition of the element. For example an air conditioner setting the room temperature to 25+ degrees celsius when the actual room temperature is less than 15 degrees Celsius.

THE DOM


The DOM (Document Object Model) is a programming interface for web documents. A web page is a document that can be either displayed in the browser window or as an HTML source. The DOM allows us to manipulate the document using a script language (JavaScript is the most commonly used language).
For example we can manipulate the HTML document to change the color of the paragraphs on the HTML file.

arrays and objects


An array in javascript is a collection of multiple data types in a single place in order. It can store different data types like a selection of strings, integers and more.
And an array of objects stores a sequence of numbered objects at a single place.

functions


A javascript function is a block of code that allows us to perform the same code without having to rewrite the same code again. It makes our code cleaner, and smoother. Functions are very important because they enable code organization, reusability, and abstraction, making it easier to develop and maintain.
